
# ROS 2 Goal Seek System (Humble)

A professional ROS 2 control project demonstrating:
- Closed-loop control
- Goal seeking and waypoint following
- Parameterized navigation logic
- Launch-based orchestration

## Modes
- Part 1: Single goal seek controller
- Part 2: Waypoint follower

## Run
ros2 launch goal_seek goal_seek_system.launch.py
